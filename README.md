# adyl-s.github.io — Portfolio

Personal portfolio of **Mohammad Adil Sheikh** — AI Engineer · Multi-Agent Systems · Voice AI.

**Live:** https://adyl-s.github.io

## Stack

Pure static HTML/CSS/vanilla JS. No framework, no build step.

## Local development

Open `index.html` in a browser, or serve it:

```bash
python -m http.server 8000
```

## Deploying updates

Any push to `main` is served by GitHub Pages automatically:

```bash
git add -A
git commit -m "Update portfolio"
git push
```

Changes go live within a minute or two.
